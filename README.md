# Web Sandbox

## Startup

Run `npm i`

## Features

-   Browserslist
-   Editor config
-   ESLint
-   .gitignore and .gitattributes
-   Prettier
-   Stylelint
-   husky, lint-staged

## How to create workspace.

-   Go to the root
-   Write npm init -w (place)
-   Write npm i (name of dependencies) -w (place)
